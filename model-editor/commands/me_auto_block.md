# Auto block
<img src="images/auto-align.svg" width="50" height="50"> 

Automatically group rooms into blocks using the most common orientation of right angles in the rooms.

## Options

* **Angle Increment**

  A positive number in degrees for the angle increment between blocks. This sets an upper limit on the number of blocks given that 90 divided by this increment yields the maximum number of categories the rooms can be grouped into

* **Separation Distance**

  The distance with which rooms will be separated into distinct blocks based on proximity
