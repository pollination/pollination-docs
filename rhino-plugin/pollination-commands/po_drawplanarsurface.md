# PO_DrawPlanarSurface

Set CPlane and draw plane surfaces on it.

## Options

* **2Points**

  Draw a rectangle by 2 points.

* **3Points**

  Draw a rectangle by 3 points.

* **FreeForm**

  Draw a closed polygon. Press enter to close the polygon.
